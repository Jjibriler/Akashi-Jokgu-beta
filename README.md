---

## 1. 개요
* 모바일 게임 벽람항로의 IP를 이용한 족구 팬게임으로 평범한 족구가 아닌 격투 게임의 커맨드 요소를 접목시켰습니다.
* 전략적 완급 조절: 커맨드 시스템을 통해 공의 속도를 바꿔가며 수비와 역습을 오가는 다채로운 전략 구사가 가능합니다.
* 특수기는 강력한 위력을 자랑하지만 복잡한 커맨드 입력 실패 시 실점으로 이어질 수 있는 극단적인 재미를 제공합니다.

---

## 2. 게임 규칙
* **득점 방식**: 플레이어는 캐릭터를 조작해 공이 상대 진영의 바닥(Ground)에 닿게 만들어 득점합니다.
* 플레이어는 가운데에 위치한 네트를 기준으로 각자 정해진 왼쪽과 오른쪽 구역에서만 조작 가능합니다.

---

## 3. 조작법
* **Player 1**: `W`, `A`, `S`, `D`를 사용하여 이동 및 점프합니다.

* **Player 1**: `D` -> 잠시 중립 -> `S` -> `D`를 1초 이내에 입력하여 순간적으로 4배의 속도로 앞으로 이동할 수 있습니다.

* **Player 1**: `D` -> 잠시 중립 -> `S` -> `D`+`SHIFT키`(D와 SHIFT는 동시)를 1초 이내에 입력하여 특수기를 발동합니다.
  
* **Player 1**: `S`를 사용하여 공의 속도를 감속시킵니다.
  
* **특수기** 발동 후 2초 이내에 플레이어에게 공이 닿을 경우 공의 속도는 3배가 되어 기습 공격이 가능합니다.
  
* **Player 2**: 방향키(`←`, `↑`, `→`, `↓`)를 사용하여 이동 및 점프합니다.
  
* **Player 2**: 방향키(`↓`)를 사용하여 공의 속도를 감속시킵니다.
  
* **기타**: `ESC` (게임 종료)
