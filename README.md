# 2D 족구 게임

이곳에 게임에 대한 짧은 한 줄 설명을 적어주세요.

---

## 0. 게임 실행
* **다운로드**: 오른쪽 `Releases` 항목에서 최신 버전의 `.zip` 파일을 다운로드 해주세요.
* **실행**: 압축을 푼 뒤 `.exe` 파일을 실행합니다.
* 

---

## 1. 개요
* 모바일 게임 벽람항로의 IP를 이용한 족구 팬게임으로 현재 개발 중에 있습니다.

---

## 2. 게임 규칙
* **득점 방식**: 플레이어는 캐릭터를 조작해 공이 상대 진영의 바닥(Ground)에 닿게 만들어 득점합니다.
* 플레이어는 가운데에 위치한 네트를 기준으로 각자 정해진 왼쪽과 오른쪽 구역에서만 조작 가능합니다.

---

## 3. 조작법
* **Player 1**: `W`, `A`, `S`, `D`를 사용하여 이동 및 점프합니다.
* **Player 2**: 방향키(`←`, `↑`, `→`, `↓`)를 사용하여 이동 및 점프합니다.
* **기타**: `ESC` (게임 종료)

---
